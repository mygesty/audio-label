## 组件详细说明

### 1. 顶部工具栏

**按钮**:

| 按钮 | 类型 | 图标 | 功能 |
|------|------|------|------|
| 新建任务 | primary | ➕ | 打开新建任务对话框 |
| 批量分配 | default | 👥 | 批量分配任务 |
| 刷新 | default | 🔄 | 刷新任务列表 |
| 导出 | default | 📤 | 导出任务数据 |

**说明**:
- 批量分配按钮仅在选中任务时可用

### 2. 过滤器

**状态过滤器**:
- 选项: 全部、待处理、进行中、待审核、已完成、返工
- 默认: 全部

**优先级过滤器**:
- 选项: 全部、高、中、低
- 默认: 全部

**人员过滤器**:
- 选项: 全部、具体人员列表
- 默认: 全部

**搜索框**:
- 占位符: "搜索任务..."
- 实时搜索

### 3. 统计图表

## 交互说明

### 新建任务

1. 点击"新建任务"按钮
2. 打开新建任务对话框
3. 填写任务信息:
   - 任务名称
   - 任务描述
   - 选择音频文件
   - 设置截止日期
   - 设置优先级
   - 设置预估工时
4. 点击"创建"完成任务创建

### 批量分配

1. 勾选要分配的任务
2. 点击"批量分配"按钮
3. 打开批量分配对话框
4. 选择分配人员
5. 选择分配方式:
   - 指定分配
   - 自动分配
   - 平均分配
6. 点击"确认"完成分配

### 查看任务详情

1. 点击任务列表中的任务
2. 跳转到任务详情页 `/tasks/:id`
3. 显示任务完整信息

### 筛选任务

1. 使用过滤器筛选任务
2. 组合多个条件精确筛选
3. 实时更新任务列表

### 导出任务

1. 点击"导出"按钮
2. 选择导出格式（Excel、PDF）
3. 选择导出内容
4. 下载导出文件

```

│  [___________________________]       │
│                                     │
│  任务描述                            │
│  ┌─────────────────────────────┐   │
│  │                             │   │
│  └─────────────────────────────┘   │
│                                     │
│  选择音频                            │
│  [选择音频文件...] (已选择 5 个)    │
│                                     │
│  截止日期                            │
│  [2026-02-20]                       │
│                                     │
│  优先级                              │

```

**字段说明**:

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 任务名称 | 文本 | 是 | 任务名称 |
| 任务描述 | 文本域 | 否 | 任务详细描述 |
| 选择音频 | 多选 | 是 | 选择要标注的音频文件 |
| 截止日期 | 日期 | 是 | 任务截止日期 |
| 优先级 | 下拉 | 是 | 高/中/低 |
| 预估工时 | 数字 | 否 | 预估工时（小时） |
| 分配给 | 多选 | 否 | 分配给哪些人员 |

| 平均分配 | 将任务平均分配给选定人员 |

## 响应式设计

### 桌面端 (≥ 1024px)

- 侧边栏宽度: 240px
- 显示所有列
- 统计图表并排显示

### 平板端 (768px - 1023px)

- 侧边栏宽度: 200px
- 统计图表改为上下排列
- 隐藏"预估工时"列

### 手机端 (< 768px)

- 侧边栏折叠
- 统计图表隐藏
- 只显示核心列（任务名称、状态、操作）
- 工具栏按钮改为图标
