## 组件详细说明

### 1. 顶部导航栏

**面包屑导航**:
- 项目名称 → 音频管理

**返回按钮**:
- 点击返回到项目详情页

### 2. 顶部工具栏

**按钮**:

| 按钮 | 类型 | 图标 | 功能 |
|------|------|------|------|
| 上传音频 | primary | 📤 | 打开上传对话框 |
| 新建文件夹 | default | 📁 | 创建新文件夹 |
| 批量删除 | danger | 🗑️ | 删除选中文件 |
| 刷新 | default | 🔄 | 刷新文件列表 |

**说明**:
- 批量删除按钮仅在选中文件时可用

### 3. 过滤器

**状态过滤器**:
- 选项: 全部、待处理、处理中、已完成、失败
- 默认: 全部

**格式过滤器**:
- 选项: 全部、MP3、WAV、FLAC、OGG、M4A
- 默认: 全部

**搜索框**:
- 占位符: "搜索文件名..."
- 实时搜索

### 4. 文件夹树形导航

**根节点**: "所有文件"

**子节点**: 文件夹和文件

**图标**:
- 文件夹: 📂
- 文件: 📄

**交互**:
- 点击文件夹展开/收起
- 点击文件跳转到标注界面
- 右键打开上下文菜单

**右键菜单**:
- 打开
- 重命名
- 移动到
- 复制
- 删除

### 5. 音频文件列表

**表格列**:

| 列名 | 宽度 | 说明 |
|------|------|------|
| 选择 | 5% | 复选框 |
| 文件名 | 30% | 文件名称 |
| 格式 | 10% | 音频格式 |
| 时长 | 15% | 音频时长 |
| 大小 | 15% | 文件大小 |
| AI状态 | 15% | AI 处理状态 |
| 操作 | 10% | 操作按钮 |

**AI 状态**:

| 状态 | 图标 | 颜色 | 说明 |
|------|------|------|------|
| 待处理 | ⏸️ | `#909399` | 等待 AI 处理 |
| 处理中 | 🔄 | `#409EFF` | AI 正在处理 |
| 已完成 | ✅ | `#67C23A` | AI 处理完成 |
| 失败 | ❌ | `#F56C6C` | AI 处理失败 |

**操作按钮**:
- 查看详情
- AI 处理
- 删除

**说明**:
- 点击文件名跳转到标注界面
- 支持 Ctrl+点击多选
- 支持 Shift+点击范围选择

### 6. 分页组件

**显示**:
- 当前页 / 总页数
- 每页显示数量: 20 / 50 / 100

**按钮**:
- 上一页
- 下一页
- 页码跳转

## 交互说明

### 上传音频

1. 点击"上传音频"按钮
2. 打开上传对话框
3. 选择上传方式:
   - 拖拽文件到上传区域
   - 点击"选择文件"
   - 点击"选择文件夹"
4. 显示上传进度
5. 上传完成后添加到列表

### 新建文件夹

1. 点击"新建文件夹"按钮
2. 输入文件夹名称
3. 点击确认创建

### 批量删除

1. 勾选要删除的文件
2. 点击"批量删除"按钮
3. 显示确认对话框
4. 确认后删除文件

### 文件夹导航

1. 点击文件夹展开/收起
2. 双击文件夹进入文件夹
3. 点击面包屑导航返回上级

### 搜索文件

1. 在搜索框输入关键词
2. 实时过滤文件列表
3. 支持文件名模糊搜索

### 右键菜单

1. 右键点击文件/文件夹
2. 显示上下文菜单
3. 选择操作

## 上传对话框

### 对话框布局

```
┌─────────────────────────────────────┐
│  上传音频                    [×]   │
├─────────────────────────────────────┤
│                                     │
│  ┌─────────────────────────────┐   │
│  │                             │   │
│  │     拖拽文件到此处上传       │   │
│  │                             │   │
│  │  或点击选择文件/文件夹       │   │
│  │                             │   │
│  └─────────────────────────────┘   │
│                                     │
│  上传进度                           │
│  ┌─────────────────────────────┐   │
│  │ meeting001.mp3  ████████░░ 80%│   │
│  │ meeting002.mp3  ██████░░░░ 60%│   │
│  └─────────────────────────────┘   │
│                                     │
│  [取消]             [上传中...]    │
└─────────────────────────────────────┘
```

**功能**:
- 支持拖拽上传
- 支持批量上传
- 显示上传进度
- 支持取消上传

## 响应式设计

### 桌面端 (≥ 1024px)

- 显示所有列
- 文件夹树宽度: 240px

### 平板端 (768px - 1023px)

- 隐藏"大小"列
- 文件夹树宽度: 200px

### 手机端 (< 768px)

- 隐藏文件夹树
- 只显示"文件名"、"格式"、"操作"列
- 工具栏按钮改为图标
