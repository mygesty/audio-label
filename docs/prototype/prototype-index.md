## 交互说明

### 通用交互

- **悬停**: 所有可点击元素悬停时显示视觉反馈
- **点击**: 点击按钮/链接触发对应操作
- **表单**: 输入框聚焦时显示边框高亮
- **加载**: 操作需要等待时显示加载动画

### 标注界面交互

- **创建标注**: 在波形上拖拽选择时间区间
- **编辑标注**: 点击已有标注打开编辑面板
- **调整时间**: 拖拽标注边缘调整起止时间
- **缩放波形**: 鼠标滚轮缩放波形
- **平移波形**: 按住鼠标左键拖动平移
- **播放控制**: 点击播放按钮或按空格键
- **跳转时间**: 点击波形任意位置或标注项
- **热力图交互**: 点击热力图跳转到对应时间点

## 响应式设计

### 断点定义

| 断点名称 | 最小宽度 | 设备 |
|---------|---------|------|
| 手机 | < 768px | 移动设备 |
| 平板 | ≥ 768px | 平板设备 |
| 桌面 | ≥ 1024px | 桌面设备 |
| 大屏 | ≥ 1440px | 大屏显示器 |

### 适配策略

- **手机端**: 侧边栏折叠、简化布局
- **平板端**: 侧边栏收起、调整间距
- **桌面端**: 完整布局、最佳体验
- **大屏端**: 优化大屏显示
